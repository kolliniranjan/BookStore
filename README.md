# 📚 MERN BookStore

A full-stack BookStore web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The application allows users to browse books, manage a shopping cart, place orders, and provides an admin dashboard for managing books and orders.

## 🚀 Features

### 👤 User Features
- User Registration & Login
- JWT Authentication
- Browse Books
- Search Books
- Filter Books by Genre
- View Book Details
- Add Books to Cart
- Update Cart Quantity
- Remove Books from Cart
- Checkout & Place Orders
- View Order History

### 🔐 Admin Features
- Admin Login
- Dashboard
- Add New Books
- Edit Books
- Delete Books
- Manage Customer Orders
- Update Order Status

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kolliniranjan/BookStore.git
```

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
